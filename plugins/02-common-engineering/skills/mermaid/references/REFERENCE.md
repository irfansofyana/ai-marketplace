# Mermaid Diagram Reference Documentation

This directory contains comprehensive reference documentation for Mermaid diagram types supported by the mermaid skill.

## Available References

- **[sequence-diagrams.md](sequence-diagrams.md)** - Complete syntax for sequence diagrams including participants, messages, activations, and control structures
- **[architecture-diagrams.md](architecture-diagrams.md)** - Architecture diagram syntax with groups, services, edges, and iconify icon support
- **[flowcharts.md](flowcharts.md)** - Flowchart syntax covering node shapes, arrows, subgraphs, and styling

## When to Use Each Diagram Type

| Diagram Type | Use For |
|--------------|---------|
| **Sequence Diagrams** | Temporal interactions, API flows, authentication sequences, microservices communication |
| **Architecture Diagrams** | System architecture, cloud infrastructure, CI/CD pipelines, service relationships |
| **Flowcharts** | Process flows, decision trees, algorithm logic, workflow documentation |

See the main [SKILL.md](../SKILL.md) for quick instructions and automatic validation workflow.
